# Microcápsulas

Microcápsulas es un proyecto de texto generativo que permite a los estudiantes obtener **microcápsulas de contenido** a partir de los cursos, organizadas por temas específicos.

- **Versión actual:** `1.0`
- **Rama por defecto y de producción:** `production_zajuna`
- **Estado:** Lista para despliegue en entorno de producción



Cualquier cambio orientado a producción debe realizarse mediante *merge requests* hacia la rama `production_zajuna` una vez superadas las pruebas técnicas correspondientes.
